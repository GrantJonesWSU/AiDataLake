Application Instructions


Functionality

  • Sign-In: Use this button to log-in or make an account.

  • Upload Files: Select database type you are utilizing and upload a .SQL file. Any other file types will result in an error and be ignored. Submit the file to the server by clicking submit.

  • User History: Performs an SQL query to pull the request and output history of the user signed into the application.

  • Recent Meta: Prints the meta tags affiliated with the most recent English to SQL search.

  • GPT-3 Generic Entry: Enter plain text English questions or instructions to get information about the user's database, its schema and its data locations.

  • GPT-3 English to SQL Entry: Enter plain text English questions or instructions about data in the user's database to generate SQL statements to obtain the desired information.
